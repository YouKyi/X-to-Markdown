Markdown

Without forced breaks, a tweet's single newlines collapse into one paragraph in strict CommonMark renderers. Obsidian preserves them either way.

Collecting replies

Off exports the author’s thread alone — a single post when it is not threaded — and skips scrolling entirely, so the export is immediate. Also switchable from the caret next to the Markdown button, since it is usually a per-post decision.

Expansion controls are identified structurally, never by their English label, and a control is only clicked when its row contains no tweet, no image and exactly one button — so it cannot land on a Follow button in a “who to follow” module. Hold Alt when clicking Markdown to export immediately without scrolling.

Collection limits

Hard ceilings apply. Lower values are honoured; higher ones are clamped.

Debug

With debug on, clicking the toolbar button on an x.com tab downloads the last few raw GraphQL payloads as a JSON file — that is how test fixtures are captured.
